# Stepik auto-tests course final project

* Python
* Selenium

## Virtual environment:

* requirements.txt

Test configuration:
* conftest.py

### Page Objects in dir pages:

* main_page.py
* login_page.py
* base_page.py

Locators for main and login pages:

* locators.py

### Run tests:

To run tests in a virtual Selenium environment,  use the command: `pytest -v --tb=line test_main_page.py`
